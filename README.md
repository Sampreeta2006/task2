📊 IPL Data Analysis (EDA)
📌 Overview
This project performs Exploratory Data Analysis (EDA) on IPL datasets to understand team performance, player statistics, and match patterns.
🛠️ Tech Stack
Python
Pandas
Matplotlib
Seaborn
📁 Dataset
matches.csv
deliveries.csv
⚙️ Features
Top winning teams analysis
Top batsmen & bowlers
Toss impact on match results
Matches per season trend
Runs distribution analysis
Correlation heatmap
▶️ How to Run
pip install pandas matplotlib seaborn
python ipl_eda.py
📊 Output
Graphs saved in graphs/ folder
Insights on teams, players, and match trends
📚 Key Insights
Few teams dominate IPL wins
Top players contribute majority of runs
Toss has moderate impact
Scoring patterns are consistent
